J'art Déco

J'art Déco est une mini entreprise créée par les maçons et les métalliers de l'EREA La Rivière. Mon père, professeur principal des maçons, m'a demander de lui créer un site pour cette mini entreprise.

Le site a pour but de présenter les produits fabriqués par les élèves, ainsi que de permettre aux clients de passer commande. Il est également possible de contacter l'entreprise via le site.

Le site est en cours de développement, mais il est déjà fonctionnel. J'ai utilisé Odoo, un outil de NoCode, ce qui m'a permis de créer ce site web rapidement et efficacement.

J'art Déco propose des braséros de tables. Tous les bénéfices de l'entrprise sont reversés à l'association APAJH44.

Technologies utilisées
